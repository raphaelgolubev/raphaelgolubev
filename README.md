### Обо мне
- Я просто питер паркер
- Сейчас заинтересован в своем аналоге Excalidraw/FigJam: https://github.com/raphaelgolubev/generic
  хочу довести проект хотя бы до того состояния, когда мне самому станет удобно использовать его на постоянной основе.
  Это будет десктопное приложение на Electron (Electrobun очень сырой, другие перспективные решения предполагают знание Rust),
  в котором юзер сможет:
      - создавать фигуры и текст, соединять их стрелками, рисовать и отлично проводить время
      - писать код (возможно с помощью Monaco Editor)
      - проектировать структуру базы данных (таблицы и связи между ними)
      - плакать от потребления приложением большого количества RAM/CPU/GPU
      - и много других крутых фич
- 🎂 Мне 26 лет
- ⚡ Интересный факт: однажды на службе в армии, в условиях полного отсутствия доступа в интернет ~~(да, это еще не было нашей повседневной реальностью)~~, я написал ДМБ таймер - приложение для отсчета времени до дембеля - на языке C# в обычном блокноте.
Затем я скомпилировал приложение с помощью вшитого в `.NET Framework 3.5` компилятора, сослуживцы были в восторге 😎
